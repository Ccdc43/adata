欢迎使用 A股实时监控网站！本指南将帮助您快速上手。

## 1. 访问网站

您可以直接通过以下链接访问网站：

[https://ledone.xyz](https://ledone.xyz)

## 2. 登录系统

本站采用无密码的 **魔法链接 (Magic Link)** 登录方式，安全又便捷。

### 2.1. 发送登录链接

1.  在登录页面，输入您的常用邮箱地址。
2.  点击 **“发送魔法链接”** 按钮。

![登录页面](https://i.imgur.com/example-login.png)  *（占位图：登录页面截图）*

系统会向您的邮箱发送一封包含一次性登录链接的邮件。

### 2.2. 点击链接登录

1.  打开您的邮箱，找到来自 “A股监控” 的邮件。
2.  点击邮件中的 **“点击登录”** 按钮，或复制下方的链接到浏览器地址栏打开。
3.  链接验证通过后，您将自动登录并跳转到监控面板。

> **注意**: 登录链接有效期为 **15 分钟**，且只能使用一次。如果链接失效，请返回登录页面重新获取。

## 3. 使用监控面板

登录后，您将进入核心功能区——监控面板。

![监控面板](https://i.imgur.com/example-dashboard.png) *（占位图：监控面板截图）*

### 3.1. 添加自选股

1.  在页面上方的 **“添加自选股”** 区域，输入您感兴趣的股票代码或名称（例如 “贵州茅台” 或 “600519”）。
2.  点击 **“搜索”** 按钮。
3.  在搜索结果列表中，找到您想要的股票，点击右侧的 **“添加”** 按钮。

添加成功后，该股票将出现在下方的 “我的自选股” 列表中，并开始实时更新行情数据。

### 3.2. 查看实时行情

“我的自选股” 列表中的每张卡片都实时展示了股票的核心行情数据：

-   **当前价格**
-   **涨跌额** 和 **涨跌幅** (红色代表上涨，绿色代表下跌)
-   **今日开盘价、最高价、最低价**
-   **昨日收盘价**
-   **数据更新时间**

数据每 **3 秒** 自动刷新一次，无需手动操作。

### 3.3. 移除自选股

如果您不再关注某个股票，只需将鼠标悬停在对应的股票卡片上，点击右上角出现的 **“×”** 图标即可将其从您的自选股列表中移除。

## 4. 退出登录

在页面右上角，点击 **“退出登录”** 按钮即可安全退出当前账户。

## 常见问题 (FAQ)

**Q: 为什么我收不到登录邮件？**

A: 请检查以下几点：
   - 邮箱地址是否输入正确。
   - 邮件是否被归入“垃圾邮件”或“广告邮件”文件夹。
   - 稍等几分钟，邮件发送可能存在延迟。

**Q: WebSocket 显示“未连接”怎么办？**

A: 这可能是由于网络问题或服务器暂时故障。请尝试刷新页面。如果问题持续存在，请联系管理员。

**Q: 数据更新频率是多少？**

A: 行情数据每 3 秒自动更新一次。
